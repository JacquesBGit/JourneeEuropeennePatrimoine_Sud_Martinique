Affiche les événements des Journées européennes du Patrimoine 2024 au Sud Martinique.

== Description ==

Ce plugin crée un shortcode [jep_events] qui affiche les événements des Journées européennes du Patrimoine 2024 pour les villes du Sud de la Martinique (Le François, Rivière-Pilote, Les Trois-Îlets).

== Installation ==

1. Téléchargez le fichier zip du plugin
2. Allez dans votre tableau de bord WordPress
3. Naviguez vers Plugins → Ajouter
4. Cliquez sur "Téléverser un plugin"
5. Choisissez le fichier zip que vous avez téléchargé
6. Cliquez sur "Installer maintenant"
7. Activez le plugin une fois l'installation terminée

== Utilisation ==

Utilisez le shortcode [jep_events] dans n'importe quelle page ou article où vous souhaitez afficher les événements.
